# 자판기 만들기
Java를 이용한 자판기 생성 프로젝트

<br />

## Intro
* 사용자는 고객과 관리자로 분리
  * 고객 : 상품 구매
  * 관리자 : 재고 확인, 매출 확인, 상품 가격 설정
* 고객이 상품 구매 시 결제 시스템
  * 현금 결제와 카드 결제 가능
  * 현금 결제 : 거스름돈 계산
  * 카드 결제 : 잔액 확인
* 관리자가 관리하는 자판기의 상품
  * 음료수와 기타 상품 구분해 관리
  * 각 상품은 이름, 가격, 재고 수량 가짐
